# FIFA World Cup 2026 — Match Schedule

A live-updating World Cup 2026 schedule with match details, head-to-head history, group standings, and real-time scores.

**Live site:** https://caralmar31.github.io/World-Cup

## Features
- Full 104-match schedule (group stage + knockouts)
- Click any match for head-to-head history, team stats, key players, and recent form
- Live scores and group standings auto-update every 90 seconds during the tournament
- Filter by team or sort by date
- Works on mobile and desktop
